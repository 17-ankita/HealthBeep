# HealthBeep - Medicine Reminder App

## Project Overview
HealthBeep is a local medicine reminder application built using Python, SQLite, HTML, CSS, and JavaScript.  
It allows users to add medicines with dosage and scheduled times and view them in a real-time schedule. The app works fully offline.

## Features
- Add medicine name, dosage, and time
- Real-time schedule displayed in the browser
- Stores data locally in SQLite (`medicines.db`)
- Optional console reminders for offline alerts
- Simple, responsive frontend interface with colors

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python HTTP server (`server.py`)  
- **Database:** SQLite  
- **Optional Analytics:** Power BI (if integrated)
